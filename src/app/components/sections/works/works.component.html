Game Jam Sherbrooke 2023

Game Jam Sherbrooke 2023

En mars 2023, j'ai eu la chance de participer au Game Jam de Sherbrooke. Avec 5 de mes collègues, nous avons développé un jeu complet en moins de 48 heures. Selon la thématique «Nuit Blanche», nous avons produit The Beast of Hamshire dont je suis très fier.

J'ai occupé le rôle de développeur et de Scrum Master durant la totalité du Game Jam.

Global Game Jam 2020

Global Game 2020

J'ai eu une fin d'année 2019 très difficile. Avant de me lancer dans une réorientation de carrière, je voulais tester les eaux du développement de projet. Avec un ami, nous avons développé un jeu de table en moins de 48 heures dans le cadre du Global Game Jam 2020. Le thèmatique « Rebuild » nous a conduit vers le simulateur de colonie du 19e siècle que nous appelons Les Utopistes .

J'ai co-développé le jeu, les textes, les tests d'assurance qualité.
